https to custom protocol redirector

This page handles custom protocol redirects. Use the following URL parameters:

  • protocol - The custom protocol to redirect to (e.g., "roblox")
  • launchargs - URL-encoded arguments to pass to the protocol

Example: ?protocol=app&launchargs=some%2Fpath

Common text replacements:

  • %2F → /
  • %3A → :
  • %20 → (a space)
  • %3F → ?
  • %3D → =
  • %26 → &

An example link may look like: https://flyingbacen.github.io/?protocol=roblox&launchargs=navigation%2Fshare_links%3Fcode%3D1b87a02eb105d0419bc5f7ad71bcef31%26type%3DServer
which will redirect to roblox://navigation/share_links?code=1b87a02eb105d0419bc5f7ad71bcef31&type=Server and open your private server automatically.

When using a private server link code instead of the share code, which is more reliable, the link would look like: https://flyingbacen.github.io/?protocol=roblox&launchargs=experiences%2Fstart%3FplaceId%3D15532962292%26linkCode%3D03684619728623677095913013347115
which redirects to roblox://experiences/start?placeId=15532962292&linkCode=03684619728623677095913013347115
